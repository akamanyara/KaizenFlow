Player's Guide

Master your routine, beat the deadlines, and level up your life.

I. How to Start

1. Set the Goals

Add one-time Quests and everyday Dailies you want to work on.

2. Stay Disciplined

Complete quests, build unbreakable daily streaks, and avoid deadlines.

3. Level Up!

Earn XP points, level up your account, and track your real-life progress.

II. Rules & Economy

Quest Rewards
  • EASY +20 XP
  • MEDIUM +40 XP
  • HARD +80 XP
  • BOSS +200 XP
Dailies Streak Bonus
  • 3 Days Streak 1.1x XP
  • 7 Days Streak 1.5x XP
  • 14 Days Streak 1.75x XP
  • 21+ Days Streak 2.0x XP
Penalties
  • Missed Daily:
    -20 XP & Streak resets to 0.
  • Missed Deadline:
    Heavy XP loss based on penalty multiplier.
  • WARNING: Falling below 0 XP causes a Level Down!

III. Terminology

Quests

One-time challenges (e.g., "Complete CS50 Project") with a set difficulty level. Once completed, they are gone.

Dailies

Daily quests designed to build a routine. Complete them consecutively to build a streak and unlock massive XP multipliers.

Deadlines

The final completion date for a Quest. Late submissions or ignored deadlines will trigger a severe XP penalty.

About the Project

This application was built as a Final Project for CS50x.

The goal of this project is to gamify the daily grind towards self-improvement by completing tasks on time and maintaining daily habits. The leveling system is designed to make you enjoy the process more by visualizing your actual progress.
